# Project ICE Decision Records

Use the `ADR-NNN_Short-Title.md` convention. Accepted decisions are immutable; a later ADR may supersede one but must not silently rewrite it.

## Current decision set

- `ADR-001` through `ADR-004` are accepted governance and positioning decisions.
- `ADR-005` through `ADR-015` are proposed architecture decisions derived from `ICE-SRC-008` and `ICE-ARCH-001`.
- Proposed decisions become accepted only after the required technical, SAP integration, security, source, and executive reviews are recorded.

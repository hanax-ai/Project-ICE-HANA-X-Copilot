# Reference Architecture

The reference architecture will depict:

```text
People and enterprise channels
        -> HANA-X Copilot (Enterprise Intent Layer / System of Intent)
        -> HANA-X Agentic OS (governed execution runtime)
        -> released SAP S/4HANA interfaces
        -> SAP S/4HANA (System of Record)
```

Every published diagram must retain an editable source, SVG render, PNG render, metadata record, caption, and alt text.

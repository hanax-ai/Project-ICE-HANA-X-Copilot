# Project ICE Requirements

## Identifier families

- `ICE-REQ-NNN` — product and functional requirement
- `ICE-NFR-NNN` — non-functional requirement
- `ICE-SEC-NNN` — security and governance requirement
- `ICE-SAP-NNN` — SAP integration requirement
- `ICE-AI-NNN` — AI and agent requirement
- `ICE-UX-NNN` — user experience requirement
- `ICE-DATA-NNN` — data and memory requirement
- `ICE-OBS-NNN` — observability requirement

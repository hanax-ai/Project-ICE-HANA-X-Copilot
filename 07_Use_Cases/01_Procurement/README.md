# Procurement — Flagship Use Case

Procurement is the primary proof-of-value scenario for the HANA-X Copilot publication. It demonstrates natural-language intent, inventory checks, supplier and contract validation, purchase requisition and purchase order workflows, human approval, auditability, and SAP OData orchestration.
